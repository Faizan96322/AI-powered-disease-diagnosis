Prediction Using Symptoms


This Application reveals how symptoms can be utilized as input parameters in machine learning to produce disease predictions. On the dataset, the machine learning technique Random Forest is used to forecast the disease.

Prediction Using Reports


A Prediction Analysis Report applies built-in prediction mechanisms and sophisticated forecasting techniques on historical data for a metric to automatically compute how that metric is likely to change in the future. The result is a graph where historical data and future predictions are both plotted together.
